## Descripción

Food Store es un sistema de gestión de pedidos de comida desarrollado en Java utilizando Programación Orientada a Objetos.

El sistema permite administrar:

- Categorías
- Productos
- Usuarios
- Pedidos
- Detalles de pedidos

mediante una aplicación de consola.

---

## Tecnologías utilizadas

- Java 21
- IntelliJ IDEA
- Programación Orientada a Objetos
- Colecciones (ArrayList)
- Excepciones personalizadas

---

## Cómo ejecutar

1. Abrir el proyecto en IntelliJ IDEA.
2. Ejecutar Main.java.
3. Utilizar el menú de consola.
